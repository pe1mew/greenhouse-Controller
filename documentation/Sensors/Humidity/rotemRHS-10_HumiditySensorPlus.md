More about Rotem® Humidity Sensor

Ideal for poultry, sheep, and pig confinement houses, Rotem's Humidity Sensor for Platinum Plus Controllers provides highly accurate data on the farmhouse humidity levels. Using this data, you can determine the exact ventilation levels in the house. Equipped with a unique heating element, this sensor provides extremely accurate readings (+/- 2%). The sensing tip can be easily replaced if needed without replacing the entire unit.

The Rotem Humidity Sensor is Available With Three Outputs

    0 - 3 VDC - Supports Rotem® Controllers
    4 - 20 mA
    0 - 10 VDC - Supports other standard controllers
